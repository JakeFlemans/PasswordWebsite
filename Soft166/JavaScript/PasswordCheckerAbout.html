PasswordChecker
Explanation of how website works:

Enter your password into the text box on the main page and click the button to assess it
Upon clicking the button the six criteria will change colour to represent the quality of the password
Red = Bad Yellow = Okay Green = Good
The Hue lights will also change colour to sync up with the criteria colours

Explination of password Assessment criteria:

Length: The system judges a password based on its length, the longer the password the better
Capital Letters: The assessor checks the password to identify capital letters, having a few capital letters improves the password
Special Character: The given password is checked for special characters, a password is stronger with a small amount
Numbers: Any numbers in the password are identified, having numbers creates a stronger password
Repeating characters: Some repeating characters are expected in any password but too many will result in a weaker one
Reused Password: Reusing your own password or one commonly used by other automatically makes it a weak password
